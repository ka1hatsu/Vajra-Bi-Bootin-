# Phase 7.3 — Preparation Planner and Backend Capability Gate

Adds an explicit raw/prepared-media execution planner, a prepared ISO option,
automatic GPT/MBR and FAT32 defaults, system-tool capability detection, tests,
and a GUI planning gate.

Phase 7.3 does not execute partition-table changes. Phase 7.4 will integrate the
approved plan into the independently validating privileged helper transaction.
